How It Works

Five steps to total memory

Connect once. Synthesize everything. Give your agents exactly the context they need — with you in control.

1
Connect Platforms
Authorize via OAuth or upload your GDPR data export. We support 20+ platforms and automatically file DSAR requests on your behalf.
2
Synthesize Memory
GPT-4o classifies every data point into 5 memory types, generates embeddings, and stores them in a searchable vector graph.
3
Mint Consent NFT
Define exactly what your agent can access: platforms, memory types, query budget, and expiry. Minted on NEAR blockchain — immutable proof.
4
Agent Queries MCP
Any AI agent — ChatGPT, Claude, or your own — queries your memory via our MCP endpoint. Each query costs 0.001 USDC via Circle x402.
5
Revoke Anytime
Burn the Consent NFT on NEAR. All agent access stops instantly. The revocation is permanently logged on-chain. You stay in control.

Key Features

Built different by design

Every architectural decision is made to keep you in control and your agents well-informed.

🔐
GDPR-Legal by Design
All data collected via official OAuth APIs or GDPR Article 20 Data Subject Access Requests. You have the legal right to your own data — we automate the process.
🔗
On-Chain Consent
Consent NFTs on NEAR blockchain give you cryptographic proof of what agents can access. Revocation is instant and permanent — no middleman, no delay.
⚡
x402 Micropayments
Agents pay 0.001 USDC per memory query via Circle's x402 protocol. The first AI memory system with native micropayment economics built in from day one.
🤖
Agent-Agnostic MCP
One MCP endpoint works with ChatGPT, Claude, Gemini, or any custom agent. Your memory travels with you regardless of which AI you use.
🧬
Merkle Provenance
Every memory is tamper-evident — traced to its source via a cryptographic Merkle-DAG. Agents know exactly where each memory came from and when.
📡
EAS Attestations
Every agent query is logged as an Ethereum Attestation Service record on Base Sepolia. Permanent, public proof of every memory access event.
